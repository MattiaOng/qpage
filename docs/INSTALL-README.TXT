*************
** INSTALL **
*************

  To install Qpage you must follow this way:
    1. Go to 'configure.php' and configure itself;
    2. Run 'install.php' with your browser;
    3. Check out what printed, if it means 'Alright' you installed anything correctly otherwise try to reconfigure 'config.php' and come back to the second point;
    4. The first account registred will be with admin permission, go to 'manage.php?act=s_register' and make it.
